# Посмотри в окно

Проект "Посмотри в окно" — это веб-приложение для поиска и просмотра видео о разных городах.

## 🎯 Функциональность

- Поиск видео по городам и времени суток
- Просмотр видео в плеере
- Подгрузка дополнительных видео
- Адаптивный дизайн

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (работа с API, DOM)
- GitHub Pages для хостинга
